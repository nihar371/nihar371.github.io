LLM-powered E-commerce Product Recommendation

Developed a product recommendation system using Large Language Models (LLMs). Scraped 12,000+ products using Postman, JavaScript, and Selenium, and persisted data over sessions. Pre-processed raw data using Regex and data descriptors, and applied BERT and Sentence Transformers (all-MiniLM-L6 & all-mpnet-base) for embeddings. Deployed the solution using Flask.

Story of Formula 1 & Williams (Tableau Project)

Tableau project leveraged Tableau Prep Builder for data transformation and cleaning, including pivoting, splitting fields, and handling missing values. Advanced features such as calculated fields, interactive filters, set actions, and dynamic parameters, were used to enhance interactivity and customization. Dashboard actions enabled drill-downs between views. Custom tooltips with embedded images added context, and data blending integrated multiple sources for enriched insights.

3D Embedding Space Visualizer

Built a Streamlit-based visualizer for custom text and image embeddings. Utilized NLP libraries like Gensim, NLTK, and spaCy for text pre-processing. Transformed data using a pre-trained Word2Vec model and applied dimensionality reduction techniques like PCA, t-SNE, and UMAP for 3D projection and visualization.

Time-series Air Quality Analysis

Analyzed air quality data by examining ACF, PACF, trend, seasonality, and residuals. Determined optimal hyperparameters for ARIMA, SARIMA, and LSTM models. Compared performance of FB Prophet, MLP, SARIMA, and Vector Auto-Regressive (VAR) models, with FB Prophet and MLP outperforming others.
